# Gazebo

Ubuntu 22.04 + ROS2 + Gazebo + PX4 + Micro XRCE-DDS

1. WARNING!!! This project needs "a4_models" folder, which I have. Tell me if you want.

2. Before start.sh,
```
   chmod +x start.sh
```
Then,
```
./start.sh
```
3. Delete a4_models>models> (x500, r1_rover whatever vehicles you use)
very important!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
